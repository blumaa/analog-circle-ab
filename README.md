# The Analog Circle — Inner Circle

Recreation of the member-only Inner Circle app, a design system + component library extracted from it, and new features (profile wall, event CRUD).

## Stack
- Vite + React + TypeScript
- CSS Modules driven by design-system tokens (CSS custom properties)
- pnpm monorepo: `packages/tokens`, `packages/ui` (+ Storybook), `apps/web`
- Firebase (Firestore + magic-link Auth)

See the implementation plan for details.
